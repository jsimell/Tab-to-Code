# Tab-to-Code: Adaptive, Passage-Level AI Assistance for Qualitative Data Coding

ACCESSIBLE AT: https://jsimell.github.io/Tab-to-Code/ 
(NOTE: You need to provide your own API key to use the tool. The key is not uploaded anywhere, as the tool runs entirely in the browser.)

Qualitative data coding---the process of highlighting passages in text and annotating them with categories or descriptive notes---is a fundamental yet time-consuming part of many forms of qualitative research. While Large Language Models (LLMs) can now automate this process, such automation risks distancing the researcher from their data, as deep analytic understanding is often built through the manual act of coding. Aiming to balance efficiency with insight, we present Tab-to-Code, a human-in-the-loop qualitative coding tool that keeps researchers actively involved in the analysis. Designed as an "intelligent autocomplete" for qualitative coding, Tab-to-Code adopts the familiar "Tab to accept, Esc to decline" interaction pattern now common in AI-assisted programming. The system provides full transparency and partial customization of the under-the-hood LLM queries and uses in-context learning to adapt the highlight and code suggestions to a researcher’s specific coding style and requirements.
